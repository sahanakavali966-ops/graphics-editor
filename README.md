# Graphics Editor

A simple terminal-based graphics editor written in C.

## Features
- Draw Circles
- Draw Rectangles
- Draw Lines
- Draw Triangles
- Delete objects
- Display picture on terminal canvas

## How to Compile
```bash
gcc main.c -o graphics-editor -lm
```

## How to Run
```bash
./graphics-editor
```

## Usage
Run the program and choose from the menu:
- 1 — Add Circle
- 2 — Add Rectangle
- 3 — Add Line
- 4 — Add Triangle
- 5 — Delete Object
- 6 — List Objects
- 7 — Display Picture
- 0 — Exit
